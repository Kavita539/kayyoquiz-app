# Kayy-o-quiz
<h4 align="center">A quiz app on your favourite series</h4>

---

## Quick Start

Preview Link : https://kayyoquiz.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/Kavita539/pomodro-app.git
```

Go to the project directory

```bash
  cd kayyoquiz-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Features Available.

- Home Page 
- Explore Quiz Page
- Questions Page
- Results Page
- Authentication
  - Login
  - Sigup
  - Guest Login
- Fully Resposive
- Dark Theme Available

---

## Tech Stack

**Client:** React, Typescript, cleanCSS

**Server:** Firebase
